Assignment website
Each player will make their own D&D corner in the website I've made. This is your personal space where you can showcase your character and keep track of your stuff like spells, inventory, stats, and combat actions. Think of it as a mini toolkit for your character.

Requirements
Inventory system
Ability to add, edit, and remove custom items. Each item should have: 
Name
Description
Quantity
Special properties (optional)
Ability to add, track, and spend gold
Sorting/filtering system
Optional: Equip/unequip items

Notes
Ability to create, edit, and delete notes.
Organize notes in folders or categories.
Password protection.

Character profile
Include character design and introduction for players.
Optional sections for:
Backstory
Goals or personality traits
Secret mode:
Hidden lore
Secret stats or information

Spell system
Full spell lists for your character.
Track spell slots, spell preparation, and spell usage.
Include spell descriptions, effects, components, and duration.

Combat features
Track HP, AC, stats, temporary bonuses, conditions. 
Include all your combat actions and feats.
Ability to record past encounters.

Other
Make the website visually engaging and interactive.
Add features like drag and drop for easier usage.
Add any other feature you can think of!


Side for non-programmers:
All you have to do is front-end. Consult someone for assistance with back-end.
